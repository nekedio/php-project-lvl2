# php-project-lvl2

### Install
1. clone repository
2. > make install

### Install globally via composer

1. Install globally composer
2. > composer global require nekedio/php-project-lvl2

### Help
> gendiff --help

### Compare two JSON
> gendiff --format json firstFile.json secondFile.json
